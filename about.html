Six Interesting Things

We discovered how to map values from one sql query to another in order to dynamically populate our table

Learned the diverse ways the Nominatim API can interact with the Leaflet API

We discovered we needed to change our Rest Server in order to correctly display the Incidents Table

While looking more closely at the data we found that a majority of incidents are proactive police visits

We found the setView method which is a great way to adjust your leaflet maps zoom and position

We learned are able to create markers on your leaflet map using json key value pairs and the Nominatim API


Tools Used

Foundation Framework
Foundation allowed us to style the HTML side of the website.

Nominatim API
The Nominatim API allowed us to search for a location by entering a location and then it converting to latitude and longitude.

Leaflet API
The Leaflet API allowed us to create a customised map with neighborhoods and crimes in said neighborhoods.

St. Paul Crime API
The St. Paul Crime API allowed us to use real crime data from St. Paul from August 2014 to December 2023.
